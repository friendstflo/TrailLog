\# TrailLog



\*\*Official Wilderness Trail Reporting App\*\*  

Built for \*\*The Mountaineers – Everett Lookout Trail Maintenance Committee\*\*  

https://www.mountaineers.org/locations-lodges/everett-branch/committees/everett-lookout-trail-maintenance-committee



\### Features

\- Superior OpenStreetMap wilderness trails (far better than Google for backcountry)

\- Drop pin + photo on long-press (Log, Brush, Treadwork, Other)

\- Real-time Firebase sync \& sharing with entire crew

\- Offline-first (caches tiles + reports)

\- Year-end Stats dashboard

\- One-tap CSV export for \*\*Landowner\*\*

\- Custom photo thumbnail info windows



\### Quick Start

1\. `git clone https://github.com/yourusername/traillog.git`

2\. Open in Android Studio (Koala 2024.1+)

3\. Add your `google-services.json` (Firebase)

4\. Download offline MBTiles map for your area and place in `/files/osmdroid/`

5\. Run on device (Android 7.0+)



Built with love using AI for the greater good of our trails. 🌲🏔️
